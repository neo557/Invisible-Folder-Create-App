# Invisible-Folder-Create-App
見られたくないものを隠して保存でき、パスワードとIDでカギを掛けれるアプリです。
＃＃＃使用方法＃＃＃
1．アプリを開きます。
すると、以下のような状況になるため、まずは新規作成ボタンをクリックしてください。

<img width="669" height="335" alt="スクリーンショット 2025-12-08 170641" src="https://github.com/user-attachments/assets/76fd0d68-4d99-4f85-b6fe-4fcd0775d822" />

次のようなウィンドウが出てくるため、好きなパスワードとIDを入力してください。

<img width="666" height="336" alt="スクリーンショット 2025-12-08 170651" src="https://github.com/user-attachments/assets/775f8d80-4626-4ab0-b3ce-982fcda1c6ec" />

2．フォルダ閲覧
アンロックをクリックすると、次のウィンドウが出てくるので、先ほどのパスワードとIDを入力してください。

<img width="667" height="334" alt="スクリーンショット 2025-12-08 170714" src="https://github.com/user-attachments/assets/1ab182cb-bc0a-4466-9e00-31b00b0919f7" />

すると、”Vaultをアンロックしました。Exprolerで開きます”と出てきます。少し待ってあげると、エクスプローラー上に外からは閲覧できないフォルダを作成できます。

<img width="664" height="337" alt="スクリーンショット 2025-12-08 170733" src="https://github.com/user-attachments/assets/df38d5f7-fddd-4fb1-b4de-0a30d4b528e7" />

3．特殊キー
初期画面で、再ロックボタンをクリックすると、一見アプリが閉じているように見えますが、裏で動いています。
これをShift+Ctrl+Lキーを押すことで、UnlockFormというウィンドウが出てきます。

<img width="475" height="328" alt="スクリーンショット 2025-12-08 170756" src="https://github.com/user-attachments/assets/3a9d602c-dd89-41be-a196-43ed55ba3a41" />

ここに”awakeme"と入れてエンターキーまたはbtnUnlockをクリックすると、パスワード入力画面が出てきます。
1．で設定したパスワードを入力してあげると、同一フォルダを開くことが可能です。
また、複数フォルダを違うパスワード・IDで使い分けることも可能です。
他人に見られたくないナニかを隠したい皆さんにぜひ。
