# Invisible-Folder-Create-App
見られたくないものを隠して保存でき、パスワードとIDでカギを掛けれるアプリです。
＃＃＃使用方法＃＃＃
1．アプリを開きます。
すると、以下のような状況になるため、まずは新規作成ボタンをクリックしてください。

次のようなウィンドウが出てくるため、好きなパスワードとIDを入力してください。

2．フォルダ閲覧
アンロックをクリックすると、次のウィンドウが出てくるので、先ほどのパスワードとIDを入力してください。
すると、”Vaultをアンロックしました。Exprolerで開きます”と出てきます。少し待ってあげると、エクスプローラー上に外からは閲覧できないフォルダを作成できます。
3．特殊キー
初期画面で、再ロックボタンをクリックすると、一見アプリが閉じているように見えますが、裏で動いています。
これをShift+Ctrl+Lキーを押すことで、UnlockFormというウィンドウが出てきます。
ここに”awakeme"と入れてエンターキーまたはbtnUnlockをクリックすると、パスワード入力画面が出てきます。
1．で設定したパスワードを入力してあげると、同一フォルダを開くことが可能です。
また、複数フォルダを違うパスワード・IDで使い分けることも可能です。
他人に見られたくないナニかを隠したい皆さんにぜひ。
